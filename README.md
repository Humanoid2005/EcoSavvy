## Run Locally

Clone the project

```bash
  git clone https://github.com/Humanoid2005/EcoSavvy.git
```

Go to the project directory

```bash
  cd EcoSavvy/Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
